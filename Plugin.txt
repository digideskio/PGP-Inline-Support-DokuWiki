===== Installation =====

Install the plugin over the webfrontend by using the url:

https://github.com/user9209/PGP-Inline-Support-DokuWiki/archive/master.zip

===== How to use =====

Use Mailvelope (https://www.mailvelope.com):

  - Generate a Key-Pair
  - Add current tab to "Authorized Domain" (do NOT use API)
  - When editing a page press Mailvelope-Button and encrypt text
  - Save page as normal
  - See Mailvelope-Button on page
  - Press decrypt and enter your password

Without this Plugin, step will 6 fail.
